# TTS - STT

Very simple example to use TTS and STT.

## TTS - Text to Speech

Converts **text input** into **voice output**.

API Reference: [SpeechSynthesis](https://developer.mozilla.org/en-US/docs/Web/API/SpeechSynthesis)

## STT - Speech to Text

Converts **audio input** into **text output**.

API Reference: [SpeechRecognition](https://developer.mozilla.org/en-US/docs/Web/API/SpeechRecognition)

---

# HOW TO USE IT

Open `index.html` in Chrome or Edge browsers. *SpeechRecognition API not supported on other browsers*.
